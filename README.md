# Kanban Board - Built with React.js

[![React Version](https://img.shields.io/badge/React-18.2.0-blue.svg)](https://reactjs.org/)

Ini adalah aplikasi Kanban yang dibuat dengan menggunakan React.js, yang memungkinkan Anda untuk mengatur task menggunakan Kanban Board ini. Aplikasi ini dirancang untuk membantu Anda mengelola task-task Anda dengan lebih efisien, melacak kemajuannya, dan dengan mudah memvisualisasikan alur kerja.

## Features

- Create, edit, dan delete tasks.
- Memindahkan task ke kolom status yang berbeda (To Do, In Progress, Complete).
- Terdapat tingkat prioritas task (Low, Medium, High).
- Menetapkan kategori dan warna agar lebih mudah dalam menkoordinasi task.
- Support dark mode.

## Getting Started

1. Clone the repository: `git clone https://github.com/rezakurniawan88/kanban-board.git`
2. Install dependencies: `npm install`
3. Start the development server: `npm start`

## Technologies Used

- React.js untuk user interface.
- React-dnd untuk drag-and-drop pada card.
- React-hook-form untuk mengelola form.
- React-hot-toast untuk menampilkan toast.
- TailwindCSS untuk styling.
- Local storage untuk persistensi data.
- Zustand untuk state management.
